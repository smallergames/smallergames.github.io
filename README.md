# Appendix N Starter

A dark-mode Vite starter using Bun + React + TypeScript + Framer Motion + Biome.

## Setup

```bash
bun install
bun run dev
bun run build
bun run lint
bun run format
```

## Scripts

- `bun run dev` — start Vite dev server
- `bun run build` — typecheck + production build
- `bun run lint` — run Biome checks
- `bun run format` — run Biome formatting
